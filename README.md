# JS - FizzBuzz
This is the twelfth exercise I have completed as part of the web development master. It focuses on loop logic and conditions in JavaScript (Multiples and divisibility).
Repository name: `js-fizzbuzz`

## 📝 Task
Write a program that prints numbers from 1 to 100. However, for multiples of 3, print “Fizz” instead of the number, and for multiples of 5, print “Buzz”.  
For numbers that are multiples of both 3 and 5, print “FizzBuzz”.
